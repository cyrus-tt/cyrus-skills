# AI Mirror — 用真实数据蒸馏你自己

> **你可以对所有人撒谎，你可以对自己撒谎，但你没法对你的行为数据撒谎。**

---

## AI 执行指令（本节面向 AI agent，非用户阅读）

当本 skill 被触发时，AI 必须严格按以下流程执行：

### 执行流程

```
1. 扫描用户提供的文件夹，按类型分类：
   - 对话类：.txt/.md 聊天记录、转写、AI 对话导出
   - 日记类：日志、备忘录、复盘、计划
   - 行为类：屏幕时间截图、日历导出、git log、运动数据
   - 简历类：简历的不同版本

2. 统计材料数量和覆盖场景，报告给用户：
   "找到 N 份材料，覆盖 X 种场景。预计分析深度：[基础版/完整版/行为审计版]。"
   - 基础版：< 5 份，只有单一场景 → Layer 1-3
   - 完整版：5+ 份，覆盖 2+ 种场景 → Layer 1-6
   - 行为审计版：有行为数据 → Layer 1-7

3. 逐层分析，每层完成后输出该层结果，等用户确认再继续下一层。
   不要一次性输出所有层。

4. 每层的每个发现必须附上原文引用（标注来源文件名）。
   禁止编造、推测或使用通用描述。没有证据的发现不输出。

5. 材料不足时的降级规则：
   - 只有 1 种场景的对话 → Layer 1 跳过"人格切换地图"，注明"需要更多场景"
   - 没有拒绝/选择类材料 → Layer 2 降级为"初步推测"并标注置信度低
   - 没有高压场景 → Layer 3 跳过"被质疑时的反应"
   - 只有跟 1 个人的对话 → Layer 4 跳过，注明"需要跟不同关系的对话"
   - 没有行为数据 → Layer 7 跳过，提示用户如何补充

6. 全部层完成后，生成 Mirror Report + 自我认知卡片。
```

### 输出风格约束

- 直接、不绕弯，像朋友说真话不像咨询师打太极
- 发现不舒服的真相时不要软化——用户来这里就是为了被"照"
- 每层必须有一个"后背发凉时刻"——最出乎用户意料的那个发现，加粗标注
- 禁止使用"赋能""接住""帮您梳理"等 AI 腔
- 引用用户原话时用 `>` 引用格式，标注文件名

## 用户指南

### 这是什么

一套让 AI 从你的**真实对话和行为数据**中，蒸馏出"你到底是什么样的人"的方法论。

不是问卷，不是星座，不是 AI 编故事。输入是你亲口说过的话、实际做过的事——输出是你可能**自己都没发现的行为模式、决策逻辑和盲区**。

### 为什么做这个

所有性格测试的输入都是**你觉得自己是什么样**。问题是——人对自己的认知天然有偏差。你说你每天运动，其实一周去一次。你说你在均衡推进五件事，数据显示一件占了 60%，有一件不到 2%。

AI Mirror 不问你"你觉得你是什么型"。它看你**说了什么、做了什么**，然后告诉你：这两者之间的差距有多大。

### 三个独有的东西

**1. 行为审计层——没有任何性格工具做这个**
把你的屏幕时间 / 日历 / commit 记录跟你嘴上说的做对比，输出一张"说 vs 做"差距表。

**2. 不给你贴标签**
你不是"INFJ"或"射手座"。你是"一个在面试时接受质疑不反击、但回家写复盘文档时判断力极准的人"。完全个人化，引用你自己的原话。

**3. 可迭代**
星座算一次就完了。AI Mirror 能做时间线对比（三个月前的你 vs 现在的你）和 Bug 追踪（上次发现的盲区改了没有）。

### 跟其他 AI 工具的区别

| | MBTI / 星座 | 数字分身类 | **AI Mirror** |
|---|---|---|---|
| 目标 | 给你贴标签 | 造一个像你的 AI | **帮你看清真实的你** |
| 输入 | 问卷 / 出生日期 | 聊天记录 | 聊天记录 **+ 行为数据** |
| 输出 | 16 种 / 12 种类型 | 模仿你说话的 AI | 盲区、Bug、说和做的差距 |
| 独有能力 | 无 | 无 | **行为审计** |
| 迭代 | 一次性 | 一次性 | **时间线对比 + Bug 追踪** |

## 准备材料（越多越准，最少 3 份）

**高价值**：
- 微信聊天记录导出（微信电脑版 → 联系人 → 导出聊天记录为 .txt）
- 面试录音转文字（飞书妙记 / 讯飞听见 / 通义听悟，免费额度够用）
- 工作会议录音/纪要
- 采访/访谈转写
- 跟 AI 的聊天历史（ChatGPT: Settings → Export / Claude: 直接复制）

**中等价值**：
- 日记 / 备忘录 / 朋友圈文字
- 简历的多个版本（对比你怎么描述自己）
- 计划 / 复盘 / 年终总结

**行为数据（解锁杀手锏 Layer 7）**：
- 手机屏幕时间报告截图（iOS: 设置→屏幕使用时间 / Android: 数字健康）
- 日历 / 时间记录 app 导出
- Git commit 历史（`git log --format="%ai %s" --since="2024-01-01" > commits.txt`）
- 运动 / 健康 app 数据导出

### 数据安全说明

你的材料内容会作为对话的一部分发送到 AI 服务提供商（如 Anthropic）的服务器进行处理。请注意：
- **不要提供**银行卡号、身份证号、密码等敏感信息
- 聊天记录中涉及他人隐私的部分，建议先脱敏（用 A/B/C 替代真名）
- 分析完成后，可以 `/clear` 清除当前会话上下文
- Anthropic 的数据政策：API 调用的数据不会用于模型训练（详见 anthropic.com/privacy）

## 6+1 层蒸馏框架

### Layer 1：表达 DNA — 你怎么说话

> 来源：聊天记录、对话转写
> 最低要求：至少 1 份对话

**AI 提取指令**：
1. 统计用户的 10 个高频口头禅/语气词，按频次排序
2. 判断句型特征（短句型/长句型/列举型/思考外显型）
3. 提取比喻习惯（用生活锚点 vs 抽象隐喻 vs 几乎不用比喻）
4. 反向画像：用户**绝不会说的话**（从未出现的表达类型）
5. 如有多场景对话：生成「人格切换地图」——不同场景下的风格差异

**后背发凉时刻**：大多数人以为自己"一直是同一个人"，但分析完会发现——你对老板说话的那个你、对朋友说话的那个你、对 AI 说话的那个你，几乎是三个不同的人。

---

### Layer 2：决策逻辑 — 你怎么选择

> 来源：人生选择、工作决策、拒绝/接受的机会
> 最低要求：材料中包含至少 2 个选择/决策场景

**AI 提取指令**：
1. 从用户的选择中逆推**真实优先级排序**（钱/成长/自由/安全/认可）
2. 列出用户拒绝过的事及理由（拒绝比接受更暴露价值观）
3. 提取隐含的机会筛选框架（用户不知道自己有，但选择模式能暴露）
4. 如有复盘类材料：验证用户直觉的准确率

**后背发凉时刻**：你以为钱对你最重要，但你每次都选了"更有趣"而不是"更赚钱"的那个。你的嘴和你的脚在投不同的票。

---

### Layer 3：情绪模式 — 什么触发你什么反应

> 来源：高压场景（面试、争论、被拒绝、被表扬）
> 最低要求：材料中包含至少 1 个高压/冲突场景

**AI 提取指令**：
1. 绘制触发器地图（兴奋/焦虑/退缩/愤怒 各自的触发器）
2. 分析被质疑时的本能反应模式
3. 如有多时间段材料：识别情绪周期（冲刺→撞墙→恢复的节奏）
4. 判断焦虑处理方式（行动/逃避/转移/建系统/写计划/购物）

**后背发凉时刻**：你以为你"抗压能力很好"，但 AI 发现你焦虑的时候不是在解决问题——是在同时开 5 件事来制造"我很忙 = 我在进步"的幻觉。

---

### Layer 4：关系模式 — 你跟不同人的默认姿态

> 来源：跟不同关系层级的人的对话
> 最低要求：至少有跟 2 个不同人的对话（否则跳过本层）

**AI 提取指令**：
1. 分别分析对权威/平辈/陌生人的姿态
2. 识别用户的「交换模式」（先给/先要/对等/过度付出）
3. 分析信任建立策略（用户可能没意识到自己有策略）
4. 如有商业/合作对话：分析谈判姿态和收费意识

**后背发凉时刻**：你以为你"很真诚"，但 AI 发现你的"真诚"是一种无意识策略——你用主动暴露弱点来换取对方的信任。

---

### Layer 5：盲区与 Bug — 你不知道的关于你自己的事

> 来源：交叉验证所有材料
> 最低要求：5+ 份材料（否则本层置信度低，标注提示）

**AI 提取指令**：
1. 交叉比对不同材料中的矛盾（对 A 说的和对 B 说的不一样）
2. 提取反复犯的错（同类模式出现 2+ 次 = Bug，标注出现次数）
3. 识别"自以为是优点但实际是弱点"的东西
4. 分析自我欺骗模式（给逃避行为找理由的固定话术）

**后背发凉时刻**：你建了一套完美的学习系统——笔记模板、分类目录、自动收集管道。AI 看完说：笔记区是空的。你在用"搭系统"来逃避真正该做的事。

---

### Layer 6：核心叙事 — 你给自己讲的人生故事

> 来源：综合所有材料
> 最低要求：10+ 份材料或时间跨度 3+ 月

**AI 提取指令**：
1. 提炼用户的自我叙事原型（逆袭/受害者/探索者/叛逃者/被错置的天才）
2. 识别用户追求成为什么人 + 害怕成为什么人
3. 挖掘最深的恐惧（不是嘴上说的，是材料暴露的）
4. 找到"嘴上说不在乎但行为说明很在乎"的矛盾
5. 识别精神图腾（反复引用的人/框架/概念）

**后背发凉时刻**：你嘴上说追求自由和创业，但所有行动都在追求别人的认可——学历、title、大公司的名字。你追求的不是自由，是"不被评价"。

---

### Layer 7（杀手锏）：行为审计 — 你说的 vs 你做的

> 来源：行为数据（屏幕时间、日历、commit 历史等）
> 最低要求：至少 1 份行为数据 + 对话中有相关声称（否则跳过）
> **这一层是 AI Mirror 独有的。**

**AI 提取指令**：
1. 对比用户在对话中声称的时间/精力分配 vs 行为数据显示的实际分配
2. 列出"空头支票"——声称要做但数据显示从未开始的事
3. 从时间戳推断真实作息模式
4. 分析注意力模式（深度工作 vs 频繁切换）
5. 用对比表输出最大的 3 个"说-做差距"

**后背发凉时刻**：你说你每天花 2 小时学英语。屏幕时间数据显示——平均 11 分钟，其中 8 分钟在翻评论区。

### Layer 2: ...

## 常见问题

**Q: 材料太少怎么办？**
最少 3 份不同场景的对话能跑 Layer 1-3。越多越准——跟面试官、朋友、家人说话的风格差异才是最有信息量的。

**Q: 数据安全？**
材料作为对话内容发送到 AI 服务商处理。建议脱敏他人姓名，不要放银行/证件等敏感信息。分析完后 `/clear` 清除会话。详见上方「数据安全说明」。

**Q: 跟心理咨询有什么区别？**
心理咨询师听你**说什么**；AI Mirror 看你**做什么**。不替代，是互补。

**Q: 结果不准怎么办？**
报告最后有 5 个场景预测。命中不到 3 个 = 材料不够或你比 AI 更了解自己。

**Q: 能分析别人吗？**
可以。喂入跟某人的对话，AI 能蒸馏对方的表达 DNA 和关系模式。准确度取决于对话量。

## 进阶玩法

**时间线对比**：
```
我有 [去年] 和 [今年] 的材料。请对比两个时期的我，哪些变了、哪些没变。
```

**关系 X 光**：
```
我有跟 [某人] 的大量对话。请同时分析我和 TA，生成关系 X 光——谁主导、谁迁就、有没有不健康的模式。
```

**Bug 追踪**：
```
上次发现我有 [某个 Bug]。这是最近一个月的新材料，检查这个 Bug 改善了没有。
```

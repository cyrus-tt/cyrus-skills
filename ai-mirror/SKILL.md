# AI Mirror — 用真实数据蒸馏你自己

> **你可以对所有人撒谎，你可以对自己撒谎，但你没法对你的行为数据撒谎。**

> 作者：Cyrus（小红书 AI 学习成长博主）
> 版本：2.0 | 2026-06-23

---

## 这是什么

一套让 AI 从你的**真实对话和行为数据**中，蒸馏出"你到底是什么样的人"的方法论。

不是问卷，不是星座，不是 AI 编故事。输入是你亲口说过的话、实际做过的事——输出是你可能**自己都没发现的行为模式、决策逻辑和盲区**。

### 为什么做这个

所有性格测试的输入都是**你觉得自己是什么样**。问题是——人对自己的认知天然有偏差。你说你每天运动，其实一周去一次。你说你在均衡推进五件事，数据显示一件占了 60%，有一件不到 2%。

AI Mirror 不问你"你觉得你是什么型"。它看你**说了什么、做了什么**，然后告诉你：这两者之间的差距有多大。

### 三个独有的东西

**1. 行为审计层——没有任何性格工具做这个**
把你的屏幕时间 / 日历 / commit 记录跟你嘴上说的做对比，输出一张"说 vs 做"差距表。这种"打脸"别的工具做不到，因为它们压根不收行为数据。

**2. 不给你贴标签**
你不是"INFJ"或"射手座"。你是"一个在面试时接受质疑不反击、但回家写复盘文档时判断力极准的人"。AI Mirror 给你的是完全个人化的模式描述，引用的是你自己说过的原话。

**3. 可迭代**
星座算一次就完了。AI Mirror 能做时间线对比（三个月前的你 vs 现在的你）和 Bug 追踪（上次发现的盲区改了没有）。你变了，你的镜子也跟着变。

### 跟其他 AI 工具的区别

| | MBTI / 星座 | 数字分身类（WeClone 等） | **AI Mirror** |
|---|---|---|---|
| 目标 | 给你贴标签 | 造一个像你的 AI | **帮你看清真实的你** |
| 输入 | 问卷 / 出生日期 | 聊天记录 | 聊天记录 **+ 行为数据** |
| 输出 | 16 种 / 12 种类型 | 能模仿你说话的 AI | 你的盲区、Bug、说和做的差距 |
| 独有能力 | 无 | 无 | **行为审计：对比你声称的 vs 你实际做的** |
| 迭代 | 一次性 | 一次性 | **时间线对比 + Bug 追踪** |

## 快速启动

### `/mirror` — 一键启动

把你的材料放在一个文件夹里，然后对 Claude 说：

```
/mirror [文件夹路径]
```

或者完整版：

```
我想用 AI Mirror 蒸馏我自己。材料在 [文件夹路径]。
按 6+1 层框架分析，每层必须引用我的原话或行为数据作为证据。
最后生成 Mirror Report + 自我认知卡片。
```

AI 会自动：
1. 扫描你的材料，识别类型（对话/日记/行为数据）
2. 按 6 层逐层提取
3. 如果有行为数据，自动解锁第 7 层（行为审计）
4. 生成完整 Mirror Report
5. 生成一张可截图分享的「自我认知卡片」

## 准备材料（越多越准，最少 3 份）

**高价值（强烈推荐）**：
- 跟朋友/同事/家人的微信聊天记录导出（`.txt`）
- 面试录音转文字
- 工作会议录音/纪要
- 你接受的任何采访/访谈转写
- 跟 AI（ChatGPT/Claude/豆包等）的聊天历史导出

**中等价值**：
- 日记 / 日志 / 备忘录
- 朋友圈 / 微博 / 小红书发帖历史
- 简历的多个版本（对比不同时期怎么描述自己）
- 你写的计划/复盘/年终总结

**行为数据（解锁"说和做的差距"——杀手锏层）**：
- 手机屏幕时间报告截图
- 日历/时间记录 app 导出
- Git commit 历史（程序员）
- 运动/健康 app 数据
- 任何能证明"你实际花了多少时间做什么"的东西

> **隐私提示**：所有数据留在你本地，不会上传任何服务器。Claude Code 在你电脑上本地运行。分析完成后数据不会被保存。

## 6+1 层蒸馏框架

### Layer 1：表达 DNA — 你怎么说话

> 来源：聊天记录、对话转写

**AI 提取的内容**：
- 你的 10 个高频口头禅（你自己可能不知道你说了多少次"其实""反正""就是"）
- 你的句子节奏（短句型？长句型？列举型？思考外显型？）
- 你的比喻习惯（用生活锚点打比方？还是用抽象隐喻？）
- 你**绝对不会说**的话是什么？（反向画像往往更准）
- **人格切换地图**：你跟不同人说话的风格差异有多大？

**后背发凉时刻**：大多数人以为自己"一直是同一个人"，但分析完会发现——你对老板说话的那个你、对朋友说话的那个你、对 AI 说话的那个你，几乎是三个不同的人。

---

### Layer 2：决策逻辑 — 你怎么选择

> 来源：人生选择、工作决策、拒绝/接受的机会

**AI 提取的内容**：
- 你做决策时**真正的优先级排序**（钱 vs 成长 vs 自由 vs 安全 vs 认可 — 排序可能跟你以为的不一样）
- 你拒绝过什么？为什么？（拒绝比接受更能暴露价值观）
- 你的机会筛选框架（你不知道你有，但 AI 能从你的选择中逆推出来）
- 你的直觉通常对不对？（从历史决策的结果验证）

**后背发凉时刻**：你以为钱对你最重要，但 AI 发现你每次都选了"更有趣"而不是"更赚钱"的那个。你的嘴和你的脚在投不同的票。

---

### Layer 3：情绪模式 — 什么触发你什么反应

> 来源：高压场景（面试、争论、被拒绝、被表扬）

**AI 提取的内容**：
- 触发器地图：什么让你兴奋 / 焦虑 / 退缩 / 愤怒？
- 被质疑时的本能反应（反击？接受？回避？先接受再事后反刍？）
- 你的情绪周期（冲刺→撞墙→恢复的节奏有多长？）
- 你处理焦虑的真实方式（行动？逃避？转移？建系统？写计划？购物？）

**后背发凉时刻**：你以为你"抗压能力很好"，但 AI 发现你焦虑的时候不是在解决问题——是在同时开 5 件事来制造"我很忙 = 我在进步"的幻觉。

---

### Layer 4：关系模式 — 你跟不同人的默认姿态

> 来源：你与不同关系层级的人的对话

**AI 提取的内容**：
- 对权威（老板/长辈）：服从型？讨好型？对抗型？表面服从内心反抗型？
- 对平辈（同事/朋友）：竞争？合作？教学？暗中对标？
- 对陌生人/新关系：信任快还是慢？暴露多还是少？
- 你在关系里的**交换模式**：习惯先给还是先要？给多了会不会怨？
- 你建立信任的真实策略是什么？（你可能没意识到你在用策略）

**后背发凉时刻**：你以为你"很真诚"，但 AI 发现你的"真诚"是一种无意识策略——你用主动暴露弱点来换取对方的信任。这在私人关系里是美德，在商业关系里是成本。

---

### Layer 5：盲区与 Bug — 你不知道的关于你自己的事

> 来源：交叉验证所有材料

**AI 提取的内容**：
- **反复犯的错**：同类错误出现了几次？你每次都觉得"这次不一样"但其实一样
- **你以为是优点但实际是弱点**的东西
- **你的自我欺骗模式**：你怎么给自己的逃避行为找理由？
- **认知-行为落差**：你"知道应该做"但反复做不到的事

**后背发凉时刻**：你建了一套完美的学习系统——笔记模板、分类目录、自动收集管道。AI 看完说：笔记区是空的。你在用"搭系统"来逃避真正该做的事。

---

### Layer 6：核心叙事 — 你给自己讲的人生故事

> 来源：综合所有材料，提炼身份认同

**AI 提取的内容**：
- 你把自己的人生讲成什么故事？（逆袭？受害者？探索者？叛逃者？被错置的天才？）
- 你追求成为什么样的人？你害怕成为什么样的人？
- 你最深的恐惧是什么？（不是你嘴上说的那个，是材料暴露的那个）
- 你有没有"嘴上说不在乎但行为说明很在乎"的东西？
- 你的精神图腾是谁？（你反复引用/提到的人或框架，暴露你的深层认同）

**后背发凉时刻**：你嘴上说追求自由和创业，但你所有的行动都在追求别人的认可——学历、title、大公司的名字。你追求的不是自由，是"不被评价"。

---

### Layer 7（杀手锏）：行为审计 — 你说的 vs 你做的

> 来源：行为数据（屏幕时间、日历、commit 历史、运动数据等）
> 这一层是 AI Mirror 独有的。没有任何性格测试能做到这个。

**AI 提取的内容**：
- **精力分配幻觉**：你声称花在各件事上的时间比例 vs 行为数据显示的真实比例
- **空头支票清单**：你说了要做但数据显示你从未开始的事
- **真实作息**：你以为你是早起型还是你实际上是？
- **注意力模式**：你是深度工作型还是频繁切换型？每次专注多久？
- **最大的说-做差距**：数据层面最震撼的那个发现

**后背发凉时刻**：你说你每天花 2 小时学英语。屏幕时间数据显示——平均 11 分钟，其中 8 分钟在翻评论区。

> **这一层之所以"最狠"**：你可以对所有人撒谎，你可以对自己撒谎，但你没法对你的行为数据撒谎。

## 常见问题

**Q: 材料太少怎么办？**
最少 3 份不同场景的对话就能跑基础版（Layer 1-5）。但越多越准——你跟面试官、跟朋友、跟家人说话的风格差异才是最有信息量的。加上行为数据能解锁 Layer 7。

**Q: 隐私安全吗？**
你的数据全程在本地电脑上，不上传任何服务器。Claude Code 本地运行。如果你分享自我认知卡片，上面只有结论，不含原始数据。

**Q: 跟心理咨询有什么区别？**
心理咨询师听你**说什么**；AI Mirror 看你**做什么**。不是替代关系，是互补。你可以把 Mirror Report 给咨询师看，比你自己描述自己准。

**Q: 结果不准怎么办？**
报告最后有 5 个场景预测。如果 3 个以上"不是你会做的事"，说明材料不够多，或者你比 AI 更了解自己（恭喜，大部分人做不到）。

**Q: 能分析别人吗？**
可以。把你跟某个人的聊天记录喂进去，AI 能蒸馏出对方的表达 DNA 和关系模式。但准确度取决于对话量和场景多样性。

## 进阶玩法

### 时间线对比
```
我有 [去年] 和 [今年] 两个时期的材料。请对比两个时期的我，告诉我哪些变了、哪些没变、变化的方向是好是坏。
```

### 关系 X 光
```
我有跟 [某人] 的大量对话。请同时分析我和 TA 的模式，生成一份"关系 X 光"——我们的互动中谁在主导、谁在迁就、有没有不健康的模式。
```

### Bug 追踪
```
上次 Mirror Report 发现我有 [某个 Bug]。这是我最近一个月的新材料，请检查这个 Bug 有没有改善。
```

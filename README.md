| Skill | 说明 | 状态 |

## 怎么用

把 `SKILL.md` 放进你的 Agent skills 目录即可：

- **Claude Code**：`~/.claude/skills/cyrus-coding-12steps/SKILL.md`
- **Codex / 支持 `~/.agents/skills/` 的工具**：`~/.agents/skills/cyrus-coding-12steps/SKILL.md`
- **Cursor**：内容并入 `.cursorrules` 或项目规则文件
- **小红书 RedSkill**：在我的笔记里点 skill 组件，复制口令粘给你的 AI 一键安装

> 提醒：口令/脚本安装等于让 AI 执行远程指令，只装可信来源的 skill。
